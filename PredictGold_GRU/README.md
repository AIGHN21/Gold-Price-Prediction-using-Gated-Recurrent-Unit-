# PredictGold_GRU

Machine Learning project scaffold.

## Structure
- `data/raw`: original data
- `data/interim`: temporary transformed data
- `data/processed`: cleaned/final data
- `data/external`: external data sources
- `notebooks`: Jupyter notebooks
- `src`: source code
- `models`: saved models
- `reports/figures`: plots and output figures
- `logs`: training logs
- `configs`: configuration files
- `tests`: unit tests
